
```bash
cd portfolio_website_basic
```

### 3. Open the project

Open the folder in **Visual Studio Code** or any code editor.

---

# How to Run the Project

Since this is a **static website**, you can run it easily.

### Method 1 — Open Directly

1. Locate the `index.html` file  
2. Double-click the file  
3. The website will open in your default browser  

### Method 2 — Using Live Server (Recommended)

1. Install **Live Server extension** in VS Code  
2. Right-click `index.html`  
3. Click **Open with Live Server**
